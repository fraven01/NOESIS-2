# Storybook Usage

Diese Sammlung dokumentiert die Seitenvorlagen der Anwendung.

- Nutze die Toolbar, um zwischen Hell/Dunkel und LTR/RTL zu wechseln.
- Das A11y-Addon sollte für Standard-Stories keine Fehler melden.
- Für visuelle Regressionstests können Chromatic oder Percy angebunden werden.
